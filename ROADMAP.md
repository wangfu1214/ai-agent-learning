# Roadmap

## Vision

AgentFlow aims to become an enterprise-oriented AI Agent framework for Java.

Instead of providing only chat capabilities, AgentFlow focuses on building reusable AI infrastructure, including prompt management, tool orchestration, model routing, memory, workflow, and retrieval-augmented generation (RAG).

The long-term goal is to help Java developers build production-ready AI applications with clean architecture and extensible design.

## Guiding Principles

- Keep business logic separate from AI infrastructure.
- Prefer extensibility over hard-coded implementations.
- Build reusable components before adding new features.
- Follow clean architecture and engineering best practices.
