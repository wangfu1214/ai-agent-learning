# AgentFlow

Enterprise AI Agent Framework based on Spring Boot and Spring AI.

## Features

- Annotation-driven Agent
- Tool Framework
- Prompt Framework
- Model Abstraction
- Spring Boot Starter

## 🚀 Quick Start

### Prerequisites

Before getting started, make sure you have:

- JDK 21
- Maven 3.9+
- Git

### Clone the project

```bash
git clone git@github.com:wangfu1214/agentflow.git
```

### Run the application

```bash
mvn spring-boot:run
```
The application will be available at:

```text
http://localhost:8080
```
